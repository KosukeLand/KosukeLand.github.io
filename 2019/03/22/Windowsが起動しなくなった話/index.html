Windowsが起動しなくなった話

BootCampでデュアルブートしたWindows10のHyper-vを有効化して再起動した．
Windowsが起動しなくなった．
つらい．．．．泣きたい．．．．

https://support.apple.com/ja-jp/HT207469

2019/03/26 追記

非公式ですが，下記の記事によるとSierra10.12.4以降であればバグが解消されている模様．
MacOSを確認すると，Sierra10.12.3だったので早速OSをMojaveにアップデート．
アップデートしてWindows10を起動したところ，無事ログインできました．

お騒がせしましたm(_ _)m

https://applech2.com/archives/20170419-hyper-v-enable-after-macos-10-12-4-update.html
